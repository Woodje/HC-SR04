HC-SR04
=======
The following illustrates the setup used in this example.
 _________________
|    HC-SR04      |
|      ___        |
|     /   \       |
|    (  R  )   GND|-> Gnd
|     \___/   Echo|-> MCUs pin 6 on port B
|      ___    Trig|-> MCUs pin 7 on port B
|     /   \    VCC|-> 5V+
|    (  T  )      |
|     \___/       |
|                 |
|_________________|

So far tested on: Atmeaga328P-PU
