HC-SR04
=======
The following illustrates the setup used in this example.
 _________________
|    HC-SR04      |
|                 |
|     /- -\       |
|    (  R  )   GND|-> Gnd
|     \- -/   Echo|-> MCUs pin 6 on port B
|             Trig|-> MCUs pin 7 on port B
|     /- -\    VCC|-> 5V+
|    (  T  )      |
|     \- -/       |
|                 |
|_________________|

So far tested on: Atmeaga328P-PU
